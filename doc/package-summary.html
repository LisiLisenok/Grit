Grit is asynchronous reactive library written on ActionScript 3. Requires Air SDK.

Grit.swc library contains following packages:

  • vm.miix.grit.collection package is a set of different collections with synchronous access. Can be used independently of the other packages
  • vm.miix.grit.emitter package - asynchronous emission of stream (or train) of values (emitter)
  • vm.miix.grit.promise package contains interfacies and general implementation of a value that may not be available yet (promise)
  • vm.miix.grit.trigger package contains interfacies and classes for trigerring operations (activate, deactivate and cancel operation or item) and triggers which alarms at some point of execution (for example, on time)
  • vm.miix.grit.wire package is intended for managing multithreading within swf. Provides capabilities to run some tasks on separated thread and exchange data between tasks in asynchronous way. Only one additional to main thread is used to run any number of tasks. Any task must run in non-blocking way to get another tasks be performed

Also usefull tasks are defined in separated swc-libraries:

  • GritImage.swc library contains vm.miix.grit.task.image package - encoding images in secondary thread
