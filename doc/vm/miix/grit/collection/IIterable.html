Packagevm.miix.grit.collection
Interfacepublic interface IIterable

iterable collection following methods may not produce new collection but just refers to the current collection (lazy operation):
  • reversed
  • sequence
  • filter
  • chain
  • map
  • range
  • skip
  • take

Use clone to produce new collection with the same items as current has. Sorting with sort function always produce new collection (eager operation).

public function skip(skipped:int):IIterable

produces a collection containing the items of this collection, after skipping the first skipped items produced by its iterator.
