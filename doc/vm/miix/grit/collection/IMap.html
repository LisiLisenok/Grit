Packagevm.miix.grit.collection
Interfacepublic interface IMap extends IIterable
Implementors Map, MapSet

Map represents a collection of Key->Item pairs, where each Key maps to a one and only one Item. All methods of iterable returns iterable of pairs, which may be not excatly map
