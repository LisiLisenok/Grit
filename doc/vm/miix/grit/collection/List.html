lock()method 
public function lock():void

lock the collection. Items removed / added to the collection are not actually removed / added, but not returned by iterators. To remove / add items call unlock. Lock / unlcok operations performed in pairs. Actual adding / removing are performed when unlock is called the same time as lock was called before.
