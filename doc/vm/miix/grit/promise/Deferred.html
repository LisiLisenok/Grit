Classpublic class Deferred

Implements IPromise, IResolver

provides an instance of the promise and manages its lifecycle. Contains promise and resolver, which provides operations to force promise transition to a resolved or rejected state.

and()method
public function and(promise:IPromise, combine:Function):IPromise

combine the current promisable with provided promise. Returns new promise that:

  • resolves when both the current promisable and the other promise are resolved with value provided by specified combine function
  • rejects when either the current term or the other promise is rejected. Rejected reason is the same as corresponding promise is rejected

Parameters

promise:IPromise — promise to be combined with this
 
combine:Function — combining function, which takes values resolved by first and second promise and returns combined value

Returns
IPromise — combined promise

onCompleted()method 
public function onCompleted(onFulfilled:Function, onRejected:Function = null):IPromise

subscribe on the promise. If promise has been resolved before subscription, appropriate callback function is called immediatly after subscription otherwise it will be called after resolving or rejecting

Parameters

onFulfilled:Function — is a callback when promise is resolved, takes value returned by promise.

If callback returns value or promise, returned promise is resolved with this, if returned nothing, promise is resolved with null

 function onFulfilled( value : * ) : *;
 
onRejected:Function (default = null) — is a callback when promise is rejected, takes rejected reason 
 function onRejected( value : * ) : void;

Returns
IPromise — promise chained with current, the promise is resolved with value returned by onFulfilled (can be another promise or emitter) or rejectedwith the same reason as current

resolve()method 
public function resolve(valueOrPromise:*):void

resolve this resolver with value value.

The value can be:

  • value - the promise is resolved with this value
  • promise - this promise will be resolved with passed promise resolved value
  • emtter - this promise will be resolved with first value resolved by emitter
