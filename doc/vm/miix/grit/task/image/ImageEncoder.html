Implements ITask

encode images task. Register server with name ImageEncodeCommand.SERVER. Listen commands ImageEncodeCommand. Receives bitmap data bytes using shared bytes specified in ImageEncodeCommand.sharedName. Encode this to type specified in ImageEncodeCommand.type and sends back using the same shred bytes.
