deploy()method 
public function deploy(qualifiedClassName:String, param:*):IPromise

deploy new task.

When deploying new instance of the task is created

Task must implement ITask

When task is completed all servers, connections and shared bytes opened within this task are closed

While all tasks deployed from this task continue to work

Parameters

qualifiedClassName:String — full name of class to be instantiated as deployed task. The class must implement ITask
 
param:* (default = NaN) — param send to task on running

Returns
IPromise — promise, which is resolved with IRegistration, when task is successfully deployed or with error reason if some error has been occured
